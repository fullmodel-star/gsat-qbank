# 📝 學測英語題庫（GSAT）

一個**離線可用、免安裝、零追蹤**的大學學測英文**歷屆題庫**練習 PWA，收錄大考中心官方題目，支援整卷模擬、題型練習、弱點分析與錯題本。與「會考題庫」共用引擎（本 App **無聽力**）。

> 純單一 HTML 檔（題目內嵌）、資料全存本機、免註冊、無廣告。

> **狀態**：✅ 已上線　｜　🩵 青色主題（theme `#0e7490`）　｜　大考中心官方 **110–115、共 281 題**、五大題型　｜　📱 響應式
> 上線網址：<https://fullmodel-star.github.io/gsat-qbank/>　｜　已收錄於[英語學習入口 hub](../14_英語學習入口/)

---

## ✨ 功能
- **五大題型**：詞彙、綜合測驗、文意選填、篇章結構、閱讀測驗
- **整卷模擬 / 題型練習 / 弱點分析 / 錯題本**（共用 cap-qbank 引擎）
- 官方 PDF 已數位化，答案經校對 **0 誤**

## 🚀 使用
瀏覽器開 `index.html` 即可；安裝成 App 需用上線 https 網址（Android Chrome →「安裝應用程式」，iPhone Safari →「加入主畫面」）。

## 📁 檔案結構
| 檔案 | 說明 |
|---|---|
| `index.html` | 主程式（題目內嵌，單檔運作） |
| `manifest.json` / `sw.js` | PWA 設定與離線快取 |
| `icon-*.png` / `apple-touch-icon.png` / `icon.svg` | 圖示 |

## 🔒 隱私
完全離線、不連外部伺服器；作答紀錄只存裝置 `localStorage`；無帳號、無追蹤、無廣告。
